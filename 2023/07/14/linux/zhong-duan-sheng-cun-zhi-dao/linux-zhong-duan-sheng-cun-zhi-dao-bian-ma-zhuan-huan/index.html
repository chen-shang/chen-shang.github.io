如何查看文件编码

  1. 通过vim查看
:set fileencoding

即可显示文件编码格式。

  1. 通过 file 命令查看
<!-- file 文件名 -->
file application-common.properties

输出如下

<!-- 文件名: Unicode text, UTF-8 text -->
application-common.properties: Unicode text, UTF-8 text

文本如何进行编解码

编码

> echo "17319271179"|base64
MTczMTkyNzExNzkK

解码

多了个-d参数

> echo "MTczMTkyNzExNzkK"|base64 -d
17319271179

文件如何进行编解码

编码

base64 -i 文件名

例如

> echo "17319271179"> a.txt
> base64 -i a.txt 
MTczMTkyNzExNzkK

解码

base64 -d 文件名
